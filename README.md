# 🚀 Telegram Web K - 智能定时签到助手 (TG-Schedule-Master)

> **专为 Telegram Web K 版本设计的全自动定时消息发送工具**

你是否为了每日的群组签到、频道保活或“进货”任务，需要每天手动发送消息？或者每次都要手动点开日历选日期感到极其繁琐？

这个脚本可以一次性帮你安排未来 **30 天**（甚至 **90 天**）的定时消息，彻底解放双手！

---

## ✨ 核心功能

* **⚡️ 极速批量进货**
只需输入一次内容，脚本自动帮你循环发送，自动翻页日历，自动选择日期。
* **🛡️ 拟人化防封机制**
拒绝死板的固定间隔！脚本内置随机延迟（`0.8秒` ~ `2.0秒`），模拟真人的犹豫操作。
* **👆 穿透级点击技术**
物理模拟点击，完美解决 Telegram Web 端按钮“点了没反应”或日历不弹出的顽固 Bug。
* **🧠 智能感知**
自动识别当前是几号，自动判断是否需要翻页到下个月，精准避开非本月的日期干扰。
* **🔧 高度客制化**
启动时弹窗询问 **“签到内容”** 和 **“执行天数”**，随用随设。

## 📖 使用教程

### 1. 安装脚本

在 Tampermonkey 中新建脚本，将代码复制粘贴并保存。

### 2. 进入界面

1. 打开 **Telegram Web K**。
2. 进入你想要自动签到的 **私聊**、**群组** 或 **频道**。
3. ⚠️ **关键步骤**：请先手动**发送一条定时消息**以进入定时消息界面，随后才能执行脚本。

> **提示**：首次使用建议先测试少量天数。

### 3. 激活光标

用鼠标 **左键点击** 一下底部的输入框，确保光标在闪烁（脚本需要知道往哪里打字）。

### 4. 启动宏

按下快捷键： <kbd>Alt</kbd> + <kbd>Q</kbd>

### 5. 配置参数

脚本启动后会依次弹出窗口：

* **弹窗 1**：输入你要发送的内容（例如 `/sign` 或 `早安`）。
* **弹窗 2**：输入你要执行的天数（例如 `30`）。

### 6. 开始挂机

脚本将自动接管操作。你会看到文字自动输入、日历自动弹出、日期自动被点击。

> [!WARNING]
> **注意**：执行期间 **请勿切换标签页** 或 **最小化浏览器**，以免影响浏览器的定时器准确性，导致脚本卡住。
